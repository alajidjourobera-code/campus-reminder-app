# Campus Reminder App

### Overview
The **Campus Reminder App** is a simple mobile application that helps university students manage their schedules, set reminders, and stay updated on campus activities.

This project is part of **CEF345: Software Development**, supervised by **Dr. Valery Nkemenyi**.

---

### Problem Statement
Many students forget important deadlines, classes, and events.  
Our goal is to provide a simple reminder tool that helps students stay organized on campus.

---

### Objectives
- Help students remember class times and campus events.  
- Provide easy-to-use reminder notifications.  
- Encourage teamwork and version control using GitHub.

---

### Proposed Solution
Develop a simple **mobile app** where students can:
- Add and edit reminders  
- View their schedule  
- Receive notification alerts
